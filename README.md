ОРИГ JAP ROM: 

Если Исо не запускается, придется патчить самому:
Патчер - https://github.com/FUComplete/Patch/releases/tag/v1.4.0
- Создает новый патченный ром + файл CONFIG.BIN
- Можно настроить конфиг в патчере, либо загрузить через тулзу в браузере и сохранить
  (CONIG.BIN лежит в $USER/Documents/PPSSPP/PSP/SAVEDATA/FUCDAT/)
  В браузере: https://fucomplete.github.io/assets/config_editor.html

ЧЕ можно делать в конфиге из интересного: 

True damage value
Minimap scale
GH drink buff
